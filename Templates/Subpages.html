  • Also I made my buttons several of times as I added more to the flowchart + more pages
  • Check out my flowchart page to see the changes I made for the subpages: Flowchart page

 

-Buttons v.s Text:

  Buttons Text
Advantages
  1. Can see the results immediately
  2. Can change colours, form, placement of each buttons without changing the code
  3. There is no code needed for Buttons- simply placing the buttons in your webpage
  1. Can edit the button code immediately
  2. No need to export/ save files
  3. No need to wait to export
  4. No need to insert and place (change the image sizes)
Disadvantages
  1. When one mistake is made, you have to go back to Fireworks and export the buttons again. And then take the file and place it in the webpage
  2. Need to copy and paste the script of the buttons- inconvient
  1. Inconvient in seeing what the button may look like
  2. Need the use of codes all the time when creating a navgation bar
  3. Hard to correct mistakes- sometimes it requires precise numbers to place the navigation bar where you want it to.
